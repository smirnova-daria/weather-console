# Weather to Excel Fetcher
A simple and efficient Python command-line utility that fetches current weather data for a specified city and saves it directly into a structured Excel file.

## Overview
This project provides a lightweight tool for quickly retrieving and storing weather forecasts. It's designed for users who need to periodically log weather data for analysis, reporting, or personal record-keeping without the overhead of a complex application. By leveraging the OpenWeatherMap API, it delivers accurate, up-to-date information directly to your spreadsheet.
